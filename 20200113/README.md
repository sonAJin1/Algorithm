## 01.이해
    - 탑에서 보낸 신호는 보낸 탑보다 높이가 높은 탑에서만 수신한다.
    - 한 번 수신된 신호는 다른 탑으로 송신되지 않는다. (중복x)
    - 오른쪽에서 왼쪽으로 탐색한다.
    - 탑은 index, 높이는 value
    - 어느 탑에서 받았는지 index를 기록하여 return
    - 신호를 수신하는 탑이 없으면 0
## 02.계획

    - 가지고 온 후 마지막 value 를 없앤다
