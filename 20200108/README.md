## 02. 계획

    filter로 index에 해당하는 값 만큼 가져온 후 fold 로 각 값을 다 더해준다
    거기에 일 수를 더해주고 %7을 했을 때 나머지 0~6일 때 각각 금요일부터 목요일까지 순서대로 적용시켜준다
